### Deployment Steps

1. **Build**: `npm run build`
2. **Commit**: Commit the `build/` directory
3. **Push**: Push to GitHub
4. **GitHub Pages**: Automatically deploys from `build/` directory

### GitHub Pages Configuration

- **Repository**: r-bsoftware/RBS_website
- **Branch**: master
- **Directory**: / (root) - serves `index.html` from build
- **Custom Domain**: redbroomsoftware.com (CNAME file)
